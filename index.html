Abstract

Vision–Language–Action (VLA) systems have advanced semantic reasoning for robots, yet achieving robust whole-body humanoid control that couples semantic understanding with dynamic adaptability remains a central challenge. Existing approaches often lack explicit modeling of high-dimensional dynamics and rely on training data that is fragmented and non-standardized, limiting generalization across domains and reliable recovery under perturbations.

In this work, we present MIRAGE, a unified mobile manipulation framework for robust humanoid control, which seamlessly integrates standardized data generation with robust execution policies. A teacher expert trained with reinforcement learning under extensive domain randomization generates large volumes of standardized trajectories, mitigating the long-standing data scarcity in embodied intelligence. Our framework introduces three key innovations: SCHEMA (Sparse Command Handling for Embodied Motion Abstraction), MIRROR (Memory-Integrated Robust Representation for Online Robustness), and RISE (Recovery via Inferred State Endpoints), which together enable generalizable training signals, robust control under domain shifts, and long-horizon recovery behaviors.

Extensive experiments show that MIRAGE consistently converts SCHEMA-formatted inputs into torque-level humanoid control and achieves strong robustness across diverse domain shifts. Quantitatively, MIRAGE reduces trajectory error by ∼17%, raises strict success by ∼16 percentage points, improves push-recovery by ∼34 points, and attains 61% fall-recovery where baselines fail. Our work advances the feasibility of resilient humanoid deployment in unstructured environments.

MIRAGE Framework

MIRAGE pipeline integrates teacher-student training, standardized data generation, and robust policy execution for whole-body humanoid control.

Three Core Innovations

SCHEMA

Sparse Command Handling for Embodied Motion Abstraction

Unified sparse-joint protocol based on SMPL-X key body joints, enabling generalizable training signals from language tokens or sparse keyframes.

MIRROR

Memory-Integrated Robust Representation for Online Robustness

Leverages recent action-state histories as implicit memory to align latent dynamics, stabilizing control under domain shifts.

RISE

Recovery via Inferred State Endpoints

Distills diverse expert recoveries into shared terminal representations, guiding long-horizon behaviors like fall-and-get-up.

Training Pipeline

Stage 1: Teacher Training

Reinforcement learning with extensive domain randomization generates diverse, standardized trajectories that mitigate data scarcity in embodied AI.

  • Domain randomization
  • Expert policy learning
  • Standardized data generation

Stage 2: Student Training

Student policy learns from teacher demonstrations through imitation learning, incorporating MIRROR and RISE for enhanced robustness and recovery.

  • Imitation learning
  • Robustness integration
  • Recovery behavior learning
